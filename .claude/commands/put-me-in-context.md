# /put-me-in-context

This command follows the shared `put-me-in-context` workflow defined in `docs/assistant_workflows.md`.

Read all files in `docs/` and produce a structured context brief covering: what this project is, current status, next steps, open decisions, and known risks. Close with "Ask me anything about this project." Then ask the user whether the brief matches their understanding and what is missing or wrong — offer to update the docs based on their answer.
